# Game Manager

用游戏术语实现知识卡片管理的卡片盒笔记法 Obsidian 插件。

## ✨ 功能特性

- **🏠 主页仪表盘**：知识管理概览，统计技能/装备/副本数量
- **⚔️ 技能（Skills）**：概念型永久笔记，专注于知识概念块
- **🛡️ 装备（Equipment）**：方法型永久笔记，侧重于方法技巧块
- **🏰 副本（Dungeon）**：闪念笔记，待整理的灵感碎片
- **👑 套装（Sets）**：项目索引笔记，聚合关联的技能和装备

## 📝 标签语法

在任意 Markdown 笔记中使用以下标签格式：

```
#skill-分类1-分类2-...-内容
#equip-分类1-分类2-...-内容
#dungeon-分类-内容
```

**规则**：最后一项为具体内容，前面所有项构建为树状层级目录。

### 示例

| 标签 | 解析结果 |
|------|----------|
| `#skill-编程-python-装饰器` | 技能 → 编程 → python → 装饰器 |
| `#equip-写作-卡片笔记法` | 装备 → 写作 → 卡片笔记法 |
| `#dungeon-今日灵感` | 副本 → 今日灵感 |

## 🎮 卡片盒笔记法映射

| 游戏术语 | 卡片盒笔记法 | 说明 |
|----------|--------------|------|
| 副本（Dungeon） | 闪念笔记 | 杂乱的灵感，待整理 |
| 技能（Skills） | 概念型永久笔记 | 成体系的知识概念 |
| 装备（Equipment） | 方法型永久笔记 | 可复用的方法技巧 |
| 套装（Sets） | 项目/索引笔记 | 主题聚合，关联多个技能和装备 |

## 📦 安装

1. 下载 `main.js`、`manifest.json`、`styles.css` 到 Obsidian 插件目录
2. 路径：`你的仓库/.obsidian/plugins/game-manager/`
3. 在 Obsidian 设置中启用插件

## 🛠️ 开发

```bash
# 安装依赖
npm install

# 开发模式（监听文件变化）
npm run dev

# 生产构建
npm run build
```
